# Prompt 1: Research Movement Ecosystem Protocols

## Objective
Research and document the real module addresses, entry functions, and view functions for all Movement ecosystem protocols that Harvest will integrate with.

## Pre-Flight Checklist
- [ ] Read `docs/issues/movement/README.md` for any Movement network issues
- [ ] Read `docs/issues/move/README.md` for any Move-related issues
- [ ] Load skill: `move-dev` before starting

## Context
Currently all protocol addresses in `frontend/constants/protocols/` are placeholders (0x000...). We need real addresses to fetch actual on-chain data.

## Tasks

### 1. Research Yuzu Protocol
Find the real deployed contract addresses for Yuzu CLMM on Movement:
- Main router module address
- Liquidity pool module
- Scripts module (for entry functions)
- Position NFT collection address

Sources to check:
- https://docs.yuzu.finance
- Yuzu Discord/Telegram
- Movement block explorer
- GitHub repos

Update `frontend/constants/protocols/yuzu.ts` with real addresses.

### 2. Research Joule Protocol
Find deployed addresses for Joule lending:
- Core lending module
- Interest rate module
- Claim/rewards module

Update `frontend/constants/protocols/joule.ts`.

### 3. Research Meridian Protocol
Find deployed addresses for Meridian DeFi:
- Staking module
- Rewards module
- Vault module

Update `frontend/constants/protocols/meridian.ts`.

### 4. Research Additional Protocols
Create configuration files for remaining 7 protocols:
